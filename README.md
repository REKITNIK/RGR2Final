# Шифровальщик AESMC

main.cpp - главный файл, управляющий всеми функциями file.cpp - файл, содержащий функции, необходимые для работы с другими файлами prompt.cpp - функция ввода пароля aes.cpp - реализация AES-128 morse.cpp - реализация Азбуки Морзе (замена бинарного кода файла символами азбуки) scytale.cpp - реализация Скиталы (сдвиг бинарного кода, согласно правилам Скиталы)
